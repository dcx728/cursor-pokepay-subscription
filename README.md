整个流程跑下来大概 15–30 分钟，比想象中快。

**第一步：通过邀请链接注册账户。** 打开 👉 [PokePay 邀请注册入口](https://bit.ly/PoKePay)，用邮箱注册。走邀请链接注册的新用户可以拿到 12.8U 的虚拟卡开卡优惠券，星际卡原价 20 美元，券后只要 7.2U；实体卡可拿 18U 券，从 88 美元降到 70 美元。这个券是真金白银能省下来的，别直接去官网裸注册。

**第二步：完成 KYC 认证。** PokePay 的 KYC 分 L1 和 L2 两级。L1 是基础认证，做一下身份信息填报就行；L2 是进阶认证，需要上传证件。建议直接做到 L2，因为实体卡和部分功能需要 L2 才能解锁。填信息的时候注意：姓名要和证件完全一致，用护照注册的话姓名一般填英文，建议用 Chrome 浏览器并打开相机权限。

**第四步：申请虚拟卡。** 在 App 内点"卡片—申请卡片"，选虚拟卡，挑你要的卡种（订 Cursor 选星际卡）。开卡费会从账户余额里扣，星际卡券后 7.2U。开卡成功后你会拿到卡号、有效期、CVV，这仨就是后面填进 Cursor 支付页的全部信息。

## 六、用 PokePay 虚拟卡订阅 Cursor Pro 的逐步教程

卡拿到了，接下来就是把 Cursor Pro 订下来。这一段是全文最关键的部分，照着做基本一次过。

1. 打开 Cursor 官网，登录你的账户，进入 Settings → Subscription，或者直接点"Upgrade to Pro"。
2. 在定价页选你要的档位。日常用选 **Pro+（$20/月）**，重度智能体用户选 **Ultra（$200/月）**。点击 Get Started。
3. 进入支付信息填写页。这里要填的是 PokePay 虚拟卡的信息：
   - 卡号：PokeCard 的 16 位 Visa 卡号
   - 有效期：MM/YY
   - CVV：App 里查到的 3 位安全码
   - 账单地址：这里很关键，**填一个美国地址**，不要填国内地址。地址要和卡 BIN 地区匹配（PokeCard 是美国 BIN），随便用一个真实的美国地址即可，州和邮编要对得上。
4. 提交支付。Cursor 会先扣 1 美元做卡验证，验证通过后扣全额订阅费。
5. 如果触发 3DS 验证（3-D Secure），按页面提示完成验证即可。PokeCard 星际卡支持 3DS，一般会跳转到一个验证页或者发一条验证码到 PokePay App。
6. 支付成功后回到 Cursor，Pro 状态会立即生效，额度会在几分钟内刷新。

整个过程的难点其实就两个：账单地址要填美国地址、3DS 验证要能过。这两点 PokeCard 星际卡都能搞定，这也是为什么它被反复推荐为 Cursor Pro USDT 卡首选。

## 七、支付失败怎么排查？这些坑我都替你踩过了

就算用了对 的卡，偶尔还是会翻车。下面是几种最常见的失败场景和对应解法。

**场景一：卡被拒，提示"card declined"。** 多半是余额不够或者账单地址对不上。先去 PokePay App 查卡余额，确保至少有订阅金额 + 1 美元验证费（Pro+ 的话至少留 22U）。账单地址一定要是美国地址，州和邮编匹配。

**场景二：提示"unsupported region"或"not available in your country"。** 这是 IP 问题。Cursor 支付页对访问 IP 有地区检测，如果你是裸连国内 IP 访问 Cursor 官网，可能直接被挡。挂一个美国节点再试。

**场景三：3DS 验证过不了。** 检查 PokePay App 通知，看有没有验证码或者验证请求没处理。如果 PokePay 端没问题，换一个浏览器再试（有人反馈 Safari + Apple Pay 组合反而更稳）。

**场景四：第一次订成功了，过几天被退款。** 这是 Cursor 风控二次审查，通常是 BIN 被标记或者地址异常。PokeCard 星际卡的 BIN 相对干净，但也不是 100%免疫。建议账单地址保持稳定，别每次换不同州。

**场景五：扣款成功但 Cursor 没刷新 Pro 状态。** 等 5–10 分钟，刷新页面，或者退出重登。还不行就联系 Cursor 官方客服，附上支付凭证。
